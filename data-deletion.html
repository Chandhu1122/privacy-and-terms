User Data Deletion Instructions

How to Request Data Deletion

Users can request deletion of their data associated with this application using the method below.

Option 1: Email Request

Send an email to:

Email: growth@vchanel.com

Email Subject:
Data Deletion Request

Include the following details in the email:

  • Facebook name
  • Registered email address
  • Application name

Your data will be deleted within 7 working days.

Option 2: Automatic Deletion

This application does not permanently store user data. Any temporary data collected is automatically deleted after usage.

Confirmation

Once the data deletion process is completed, a confirmation email will be sent to the user.

Compliance

This application complies with Meta Platform Policies and Facebook User Data Deletion requirements.
